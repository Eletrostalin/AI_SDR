Telegram Bot for Campaign Management

Этот проект представляет собой Telegram-бота для управления рекламными кампаниями и рассылками. Бот классифицирует пользовательские запросы с помощью модели, чтобы определить их намерения, и маршрутизирует запросы к соответствующим обработчикам.

Бот позволяет:
	•	Создавать и управлять рекламными кампаниями.
	•	Планировать контентные планы и волны рассылок.
	•	Фильтровать лиды для рассылок на основе параметров (например, география, контактные данные).

Бот работает в режиме Long Polling, но планируется переход Webhook

Требования к серверу:
	1.	Минимальные ресурсы:
	•	CPU: 2 ядра.
	•	RAM: 4 ГБ.
	•	Диск: 20 ГБ
	2.	Операционная система:
	•	Ubuntu
    3.  Программное обеспечение
	•	Python 3.11
	•	PostgreSQL 14.13
	•	Поддержка установки зависимостей через pip (указаны в requirements.txt)
	•	Доступ к OpenAI API Key

	Планируется:
	•	Поддержка HTTPS
	•	Сертификат SSL


Зависимости окружения хранятся в requirements.txt
Переменные окружения храняться в .env

Запуск проекта реализован через Docker.
Для запуска локально необходимо:

Сделать билд проекта -   
docker-compose build --no-cache

Запустить проект -
docker-compose up

Остановить его можно коммандой-
docker-compose down
