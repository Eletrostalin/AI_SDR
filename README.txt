AI-SDR: Telegram Bot for B2B Email Campaign Management

Описание

AI-SDR — это асинхронный Telegram-бот, позволяющий автоматизировать полный цикл создания и управления B2B email-кампаниями. Бот поддерживает:
	•	классификацию пользовательских запросов через LLM
	•	создание и хранение компаний и сегментированных списков рассылок
	•	планирование контент-планов и генерацию шаблонов писем
	•	многоступенчатую генерацию черновиков писем с помощью LangGraph и OpenAI API
	•	отслеживание статистики и управление рассылкой

Особенности
	•	Полностью асинхронная архитектура на Python 3.11 с использованием async/await
	•	Обработка состояний (FSM) для многошаговых сценариев с aiogram 3.14
	•	Хранение данных в PostgreSQL через SQLAlchemy 2.0 и динамическое управление таблицами
	•	Интеграция с OpenAI API, LangChain и LangGraph для классификации и генерации текстов
	•	Импорт и сегментация Excel-файлов контактов через pandas и openpyxl
	•	Парсинг PDF/Word документов: pdfplumber, python-docx
	•	Парсинг и веб-запросы: httpx, beautifulsoup4
	•	Контейнеризация через Docker и Docker Compose с настроенными конфигурациями БД

Технологический стек
	•	Язык: Python 3.11
	•	Telegram-бот: aiogram 3.14.0
	•	ORM и БД: SQLAlchemy 2.0, PostgreSQL (+ pgvector)
	•	AI и NLP: OpenAI API, LangChain, LangGraph, LangSmith
	•	Работа с данными: pandas, openpyxl, pdfplumber, python-docx
	•	HTTP и парсинг: httpx, requests, beautifulsoup4
	•	Документы и таблицы: gspread для Google Sheets
	•	Контейнеризация: Docker, Docker Compose




Примеры использования
	•	Добавить компанию:
Отправьте команду или запрос боту, бот запросит название и детали компании.
	•	Создать контент-план:
Перейдите в меню контент-планов, введите тему и параметры.
	•	Загрузить список рассылки:
Прикрепите Excel-файл, бот сегментирует контакты по заданным критериям.
	•	Сгенерировать шаблон письма:
Выберите шаблон и контент-план — бот создаст текст на основе LLM.
